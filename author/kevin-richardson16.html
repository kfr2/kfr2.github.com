Items of the Week: June 10, 2013 ¶

  • Obey the Testing Goat -- This website for the forth-coming Test-Driven Web Development with Python explores how one can effectively utilize Test-Driven Development while developing web applications with the Django web framework. You may view the book online to gain a better understanding of the topics it will cover.
  • Map Stack -- "Map Stack makes designing maps free, easy, and fun" and is currently available from 11am to 5pm PST (Monday to Friday). The service allows you to modify the different parts of the map stack (backgrounds, roads, labels, etc) using controls for color, opacity, brightness, and more.
  • Golang Diaries I -- Tim Bray discusses experiences he faces while learning the Go programming language. I'm also an advocate of his philosophy that "[...] a really good time to write about something is while you're still discovering it" and will strive to do so more frequently.
